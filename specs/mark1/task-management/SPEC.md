# Task Management Specification

**Inherits**: `specs/core/SPEC.md`

## Overview

Hierarchical task management with relationships, context assignment, and agent assignment.

## Relationships

- depends on
- is dependent of
- loop to
- loop from
- parallel with

## Context Assignment

- fresh context
- summarized context (field: summarization prompt - what and how to summarize)
- fork context

## Agent Assignment

- assigned agent

## Task Fields

- title
- description
- ACs (Acceptance Criteria)
- FR (Functional Requirements)
- NFR (Non-Functional Requirements)
